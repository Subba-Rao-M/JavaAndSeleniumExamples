Difference between Selenium and WebDriver:
Selenium is suite of tools to automate web browsers like webdriver, ide and grid

WebDriver: create browser based regression test suites to execute in scalable manner 

selenium webdrvier architecture

client drivers(java/python) - w3c protocol - browser driver - http - real browsers

Browser drivers are agents which communicate with w3c protocol and real browser using http protocol


**Selenium Features**

•	Selenium is open Source Automation Testing tool

•	It is exclusively for Web Based applications.

•	Selenium supports multiple browsers - 
Chrome, Firefox, Internet Explorer, Safari

•	Selenium works with Multiple Platforms 
Windows, Apple OS X, Linux

•	Selenium can be coded in multiple languages - 
 Java, C#, Python, Javascript, Python, php,Ruby
